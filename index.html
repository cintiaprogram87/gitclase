Clase Git


Pasos para hacer un archivo en gib:

1- git init: primero se debo inicializar git por unica vez, abriendo una nueva terminal y escribo "git init". Antes de inicialzar debo tener abierta unicamente la carpeta en la que estoy trabajando, sino el git se aplicara a todas las carpetas mostrdas en visual studio. 2- git status: me permite ver el estado de los archivos, aparcen en rojo cuando no estan en staging area, si lo estan aparecen en color verde. 3- git add .= paso todos los archivos que tengo en el working directory al staging area, si quiero uno en particular coloco git add y el nombre del archivo. 4- git commit -m "comentario": los cambios deben ser comitiados, es mi copia de informacion. Se pasan los archivos al repositorio local.


Agregar los archivos en github por primera vez:

1-git commit -m "segundo commit" 2-git branch -M main 3-git remote add origin https://github.com/cintiaprogram87/gitclase.git 4-git push -u origin main

Agregar los archivos modificados que ya se encuentran en github:

1- git add . 2-git commit -m "otro nombre" 3-git push
